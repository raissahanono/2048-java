# 2048 (Java Swing)

A Java Swing recreation of the classic **2048** puzzle game featuring **Undo**, score tracking, and automatic saving/loading.

## Features
- Desktop UI built with **Java Swing**
- Move tiles with the **arrow keys**
- Tiles merge when equal values collide
- **Score** + **Highest Score** tracking
- **Undo** (reverts the most recent move)
- **Auto-save / Auto-load** using `save.txt`
- Reset + Instructions

## Controls
- **Arrow Keys**: move tiles (← → ↑ ↓)
- **Undo** button: revert last move
- **Reset** button: start a new game
- **Instructions** button: shows how to play

## How to Run
1. Open the project in your IDE (IntelliJ / Eclipse)
2. Run **`Game.java`** (this file contains the `main` method and launches the Swing UI)
